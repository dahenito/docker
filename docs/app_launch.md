# docker app launch

<!---MARKER_GEN_START-->
Launch app from URL

### Options

| Name             | Type     | Default | Description                                             |
|:-----------------|:---------|:--------|:--------------------------------------------------------|
| `--destination`  | `string` | `auto`  | Set local host path for app                             |
| `-d`, `--detach` | `bool`   |         | Do not wait for app to finish                           |
| `-q`, `--quiet`  | `bool`   |         | Suppress the build output and print image ID on success |
